
## 婚礼请柬小程序
基于微信小程序做的一个婚礼请柬

## 有哪些功能？

* 相册展示
* 邀请函展示
* 地图导航
* 好友祝福
* 留言评论


## 使用说明

1.修改app.js中 globalData的appid为您自己申请的小程序appid；<br/>
2.小程序后台中配置服务器域名：https://wx.qiaker.cn;<br/>
3.开发者工具新建项目上传即可
